# Unispot - LNU Parking Solution

## Overview

Unispot is a comprehensive solution designed to streamline parking management at **Lviv National University (LNU)**. The project focuses on providing a seamless experience for students, faculty, and visitors by integrating modern technologies to improve parking efficiency.

## Objectives

1. **Analyze the current state of parking** at LNU to understand existing challenges and opportunities.  
2. **Develop a mobile application** with essential features:
   - Search for available parking spaces  
   - Navigation to those spaces  
   - Integration with existing systems (if applicable)  
3. **Implement a mobile application** with a user-friendly and responsive interface.  
4. **Conduct application testing** to ensure its proper functionality and effectiveness.  
5. **Evaluate results** and prepare proposals for further development and enhancement.

## Technology Stack

### Frontend  
- **React.js** with **Tailwind CSS** for responsive design and adaptability.

### Backend  
- **Python (Django)** for robust data handling and integration with AI services.

### AI  
- **OpenCV** combined with **TensorFlow** for real-time video stream analysis to support intelligent parking management.

### Mobile Application (optional)  
- **React Native** for seamless integration with the web application.

## Features

- **Real-time availability** of parking spaces.  
- **Navigation assistance** to parking spots.  
- **Integration** with AI models for advanced analytics.  
- **User-friendly interface** for both web and mobile platforms.
